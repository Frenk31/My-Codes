In questo progetto è stata realizzata una Web Application con il quale è possibile gestire una rubrica telefonica.

Vengono effettuate le diverse mappature delle entity con le specifiche JPA.
Viene realizzata una interfaccia grafica con i diversi Templates.
Questa applicazione viene gestita tramite il server Apache Tomee,
e il collegamento con la Persistence Context in modo che si creino in automatico le tabelle.
