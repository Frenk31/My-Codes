Questo progetto è stato realizzato con il framework Spring boot.
Tale applicazione permette di memorizzare un elenco di film(titolo, anno, regista, tipo, incasso)
e di effettuare una ricerca per regista.
L'incasso del film è criptato quindi viene utilizzato il codice BCrypt.
Il progetto e i metodi a lui associati sono documentati tutti tramite java api.
